<h1 align="center">Operating System 💻 </h1>
<h2 align="center"><i>Java Based system</i></h2>
<p>This is a Java based Operating System project with basic operating system functionalities such as Process management, Memory management, I/O management and Synchronization. </p>

<p><b><i>Process Scheduling</i></b> 
<ul>FCFS</ul>
<ul>SJFS(Pre-emptive)</ul>
<ul>SJFS(non-Pre-emptive)</ul>
<ul>Round Robin</ul>
<b><i>Memory Management</i></b>
<ul>Contiguous</ul>
<ul>Non-Contiguous</ul>
<b><i>I/O Management</i></b>
<b><i>Synchronization</i></b> 
<ul>Binary Semaphores</ul>
